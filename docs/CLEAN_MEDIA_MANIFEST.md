# Clean media pack 01

Новый пакет заменяет быстрые скриншоты слайдов на более чистые кропы фото/рендеров.

Внутри:
- `photos/<PREFIX>-N.jpg` — совместимость со старой версией сайта;
- `photos/projects/<PREFIX>.jpg` и `photos/projects/<PREFIX>-N.jpg` — обложки/проектные фото;
- `photos/units/<UNITID>-N.jpg` — галереи юнитов.

Важно: это улучшенный пакет, но не финальный production-grade media set. Для Aravita/Fairway/Surfside/Kaluna лучше отдельно достать оригинальные рендеры без дизайна презентаций.
