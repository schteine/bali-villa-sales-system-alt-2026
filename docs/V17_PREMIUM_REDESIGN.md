# v17 Premium Redesign — Bali Private Deals

## Direction
Private investment desk for curated Bali villa deals: warm luxury minimal, editorial cards, quiet motion, stronger hierarchy.

## Key changes
- Brand changed to **Bali Private Deals**.
- RU remains default, EN secondary.
- CTA changed to **Получить подборку / Get selection**.
- Catalog shows **6 primary cards in 3 columns**, then **3 blurred teaser cards**.
- Card hierarchy redesigned around **Наша оценка ROI** as the primary metric.
- New asset paths to avoid GitHub Pages cache:
  - `assets/css/styles.v17.css`
  - `assets/js/app.v17.js`
- Mobile layout is redesigned: one-card flow, larger images, sticky-feeling CTA area, readable deal room.
- Motion: soft card entrance, image hover scale, elevated micro-interactions, reduced-motion safe.
